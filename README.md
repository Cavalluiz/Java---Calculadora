# Calculadora em Java

Projeto simples de calculadora desenvolvido em Java para prática de lógica de programação.

## 📚 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## 🚀 Tecnologias
- Java

## 💻 Como executar
1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe principal
